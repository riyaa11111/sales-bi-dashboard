# 📊 Automated Sales BI & Analytics Dashboard

An interactive Business Intelligence dashboard built with Python and Streamlit that turns messy, raw transactional logs into actionable corporate insights.

## 🚀 Live Demo
https://sales-bi-dashboard-sxn86yukbvzvmmqh8rsxjl.streamlit.app/

## ✨ Key Features
- **Dynamic File Ingestion:** Supports automated processing of user-uploaded Excel spreadsheets or CSV files on the fly.
- **Executive Metric Streamlining:** Dynamically computes top-tier business metrics including Total Revenue, Volume Counter, and Average Order Value (AOV).
- **Interactive Visualizations:** Implements cross-filtering data plots tracking geographical performance, inventory velocities, and time-series growth vectors.
- **Automated Report Export Engine:** Formats and writes isolated operational metrics across secondary worksheet channels inside an downloadable analytical `.xlsx` package.

## 🛠️ Tech Stack
- **Data Engineering:** Python, Pandas, OpenPyXL
- **Interface & Visualization:** Streamlit Core UI Components
